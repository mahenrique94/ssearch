Ssearch

Download

O Ssearch é uma forma para conseguir realizar filtros nos selects do html que por padrão não possuem essa funcionalidade.

Para começar a usar basta adicionar a classe ss-search no select que desejam ser filtrado, lembrando que você já deve ter feito o download e importação do ssearch.css e ssearch.js.
